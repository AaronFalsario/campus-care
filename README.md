Campus Care is a platform designed to support student wellness, safety, and academic success within a campus environment. 
It provides students with easy access to campus resources, health services, and support systems.
----------------------------------------------------------------------------------------------------------------------------
Features
🧑‍🎓 Student Support Services – Connect students with campus assistance programs.
📢 Announcements & Notifications – Stay updated with actions of admins.
🔒 Secure Student Access – Safe and protected user authentication.
--------------------------------------------------------------------------------------------------------------------------
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Node.js / laravel / Python 
Database: Supabase
Version Control: Git
